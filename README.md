# Guess Number

1. 游戏开始后，系统会随机给出一个四位，每位都不重复的数字作为答案。由用户输入自己猜测的四个数字。

2. 系统会将两个数字进行对比，并给形出xAxB的提示， 比如”2A1B”。

3. 如果数字猜对而且位置也对，就是一个A。

4. 如果数字猜对但位置不对，就是一个B。

## 例如：

系统给出”1234”，用户输入”1234”

返回”4A0B”

系统给出”1234”，用户输入”4321”

返回”0A4B”
